### Pop-up States
| State | Condition | UI |
|---|---|---|
| Entering | 0-300ms | Scale animation 0→1, confetti starts |
| Active | 300ms-10s | Full display, confetti animating |
| Dismissing | Button click or 10s timeout | Fade out 200ms |
| Dismissed | After animation | Removed from DOM, localStorage flag set |
